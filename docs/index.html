面向回合游戏的开源游戏引擎

文档 • 变更日志 • 项目

编写简单函数, 以描述在执行特定操作时游戏状态的变化。 这会自动转换成一个完整的可玩游戏, 其中包括在线多人 游戏功能, 这一切无需您编写一行网络或存储代码。

特点

状态管理

游戏状态在客户端、服务器和存储设备之间无缝管理。

多人游戏

游戏状态实时且跨平台同步。

AI

可以自动生成可以玩游戏的机器人。

游戏阶段

每个阶段有不同的游戏规则和回合顺序。

大厅

玩家匹配和游戏创建。

原型制作

在渲染游戏之前模拟操作的界面。

可扩展

允许创建新抽象概念的插件系统。

视层无关

使用原生的 JS 客户端或者 React 的封装库。

日志

游戏日志包含时间旅行功能 (查看游戏在之前某一时刻的状态)。
